# Универсальное бургер-меню для сайта Никса

## Что было сделано

Создана универсальная система навигации с правильно работающим бургер-меню для всех страниц сайта.

## Новые файлы

### 1. `navigation.css`
Содержит универсальные стили для:
- Анимации бургер-меню (трансформация полосок в крестик)
- Полноэкранное мобильное меню
- Адаптивные стили для разных размеров экранов

### 2. `navigation.js`
Содержит универсальную логику для:
- Открытие/закрытие меню по клику на бургер
- Закрытие меню при клике на ссылку
- Закрытие меню при клике вне его области
- Закрытие меню при изменении размера экрана
- Закрытие меню по нажатию клавиши ESC
- Блокировка скролла при открытом меню
- Эффект скролла для навбара

Все HTML страницы были обновлены:

## Особенности реализации

### Мобильное меню
- Полноэкранное меню с размытым фоном
- Плавная анимация появления/исчезновения
- Центрированные ссылки с увеличенным размером шрифта
- Блокировка скролла страницы при открытом меню

### Анимация бургера
- Плавная трансформация трех полосок в крестик
- Синхронизация с открытием/закрытием меню
- Использование CSS transforms для оптимальной производительности

### Адаптивность
- Автоматическое переключение между десктопным и мобильным видом
- Оптимизация для планшетов и телефонов
- Корректная работа при изменении ориентации устройства

## Как использовать

### Для новых страниц
1. Подключите CSS файлы:
```html
<link rel="stylesheet" href="navigation.css">
```

2. Подключите JavaScript файл:
```html
<script src="navigation.js"></script>
```

3. Используйте стандартную структуру навигации:
```html
<nav class="navbar" id="navbar">
    <div class="nav-container">
        <div class="nav-logo">
            <a href="index.html">
                <img src="images/niks.png" alt="Никс - Фотограф">
            </a>
        </div>
        <ul class="nav-menu" id="nav-menu">
            <!-- Ссылки меню -->
        </ul>
        <div class="hamburger" id="hamburger">
            <span></span>
            <span></span>
            <span></span>
        </div>
    </div>
</nav>
```
